# Workforce Analytics End-to-End Project

## 📌 Project Overview

This project demonstrates a complete workforce analytics workflow using Excel, Python, SQL, and Power BI.

The objective was to analyze employee attrition patterns, workforce demographics, salary trends, and employee performance to generate actionable business insights.

The project follows a real-world analytics pipeline from raw data exploration to dashboard development.

---

# 🛠️ Technologies Used

* Excel
* Python
* Pandas
* NumPy
* SQL
* MySQL
* Power BI
* DAX

---

# 🚀 Project Workflow

## 1️⃣ Data Understanding (Excel)

The dataset was initially explored in Excel to:

* Understand dataset structure
* Identify missing values
* Review column distributions
* Generate preliminary observations

---

## 2️⃣ Data Cleaning (Python)

Data preprocessing was performed using Pandas.

Cleaning tasks included:

* Missing value treatment
* Mean, Median and Mode imputation
* Handling categorical values
* Dataset validation
* Exporting cleaned dataset

---

## 3️⃣ SQL Analysis

The cleaned dataset was loaded into SQL for analytical querying.

Analysis included:

* Attrition Analysis
* Salary Analysis
* Department-wise Analysis
* Education-wise Analysis
* Work Mode Analysis
* Experience Analysis

---

## 4️⃣ Exploratory Data Analysis (EDA)

EDA was performed using Python visualization libraries.

Visualizations included:

* Salary Distribution
* Attrition vs Salary
* Attrition vs Experience
* Correlation Analysis
* Workforce Trends

---

## 5️⃣ Dashboard Development

Business insights were visualized using Power BI.

Interactive dashboard components included:

* Attrition KPIs
* Employee Demographics
* Salary Analysis
* Department Analytics
* Workforce Trends

---

# 📊 Key Business Insights

### Employee Attrition Patterns

* Identified departments with higher employee attrition.
* Analyzed attrition across different education levels.

### Salary Analysis

* Compared salaries across departments.
* Investigated the relationship between salary and attrition.

### Workforce Demographics

* Studied employee distribution by education and work mode.
* Analyzed workforce composition across departments.

### Experience Analysis

* Explored the relationship between employee experience and attrition.

### Performance Insights

* Evaluated performance-related workforce trends.
